## 📱 АРХИТЕКТУРА

### Страницы
- **index.html**: главная с категориями вакансий
- **settings.html**: настройки с вкладками (ключевые слова + каналы)
- **favorites.html**: избранные вакансии

### JavaScript модули
- **theme-manager.js**: управление темами на всех страницах
- **api-service.js**: работа с Supabase API
- **state-manager.js**: состояние приложения
- **dom-manager.js**: манипуляции с DOM

### CSS
- **style.css**: основные стили + темная тема
- **retro-settings.css**: стили страницы настроек
- **critical.css**: критические стили для быстрой загрузки
