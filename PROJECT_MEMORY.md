## 🌙 ТЕМНАЯ ТЕМА (ОБНОВЛЕНА)

**Базируется на**: Material Design 2024 + лучшие практики  
**Фон**: #121212 (не чистый черный!)  
**Карточки**: #1E1E1E  
**Текст**: #E8E8E8 (мягкий белый)  
**Акценты**: десатурированные (-20% saturation)

**Управление**: ТОЛЬКО из настроек  
**Охват**: все страницы через theme-manager.js  

## 📱 АРХИТЕКТУРА

### Страницы
- **index.html**: главная с категориями вакансий
- **settings.html**: настройки с вкладками (ключевые слова + каналы)
- **favorites.html**: избранные вакансии

### JavaScript модули
- **theme-manager.js**: управление темами на всех страницах
- **api-service.js**: работа с Supabase API
- **state-manager.js**: состояние приложения
- **dom-manager.js**: манипуляции с DOM

### CSS
- **style.css**: основные стили + темная тема
- **retro-settings.css**: стили страницы настроек
- **critical.css**: критические стили для быстрой загрузки

## ⚙️ КАНАЛЫ В НАСТРОЙКАХ

**ВАЖНО**: Активные каналы в 2 колонки БЕЗ скролла!  
- Десктоп: 2 колонки, gap 8px
- Мобильные <480px: 1 колонка для читаемости
- CSS: `.channels-list { grid-template-columns: 1fr 1fr; }`
- Компактный дизайн для экономии места
