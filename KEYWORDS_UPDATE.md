### 1. Новый ретро-пиксельный дизайн

- Удаление всех слов с подтверждением
- Автосохранение при изменениях
- Загрузка пресетов одним кликом

### 4. UX улучшения
- Hover эффекты для кнопок и тегов
- Анимации появления модального окна

- Enter для быстрого добавления

## Структура интерфейса

```
1. 🔑 КЛЮЧЕВЫЕ СЛОВА [0] слов активно
2. ➕ ДОБАВИТЬ
3. Одно слово или несколько через запятую:
4. [Поле ввода] [+]
5. [Добавить] [Удалить] [Загрузить пресет]
6. ⭐ АКТИВНЫЕ КЛЮЧЕВЫЕ СЛОВА:
7. [тег] [×] [тег] [×] [тег] [×]
8. [Сохранить как пресет] [Удалить все]
```

## Технические детали

### CSS классы
- `.keywords-header-card` - заголовок секции
- `.keywords-add-card` - секция добавления
- `.keywords-active-card` - активные слова
- `.presets-modal-overlay` - модальное окно
- `.keyword-tag` - теги ключевых слов

### JavaScript функции
- `showPresetsModal()` - показать пресеты
- `loadPreset(type)` - загрузить пресет
- `saveAsPreset()` - сохранить пресет (заглушка)
- `addKeyword(keyword)` - добавить слово
- `removeKeyword(keyword)` - удалить слово

### Заглушки
- Сохранение пресетов (показывает prompt)
- Сохранение ключевых слов (логирует в консоль)
- Загрузка ключевых слов (тестовые данные)

## Как использовать

1. **Добавить слово**: введите в поле и нажмите "Добавить" или Enter
2. **Загрузить пресет**: нажмите "Загрузить пресет" и выберите категорию
3. **Удалить слово**: нажмите × на теге
4. **Удалить все**: нажмите "Удалить все" и подтвердите
5. **Сохранить пресет**: нажмите "Сохранить как пресет" (заглушка)
