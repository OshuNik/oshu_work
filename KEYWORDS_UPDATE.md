## Структура HTML
```html
<div class="keywords-window-header">
  <span class="keywords-window-title">КЛЮЧЕВЫЕ СЛОВА</span>
  <div class="pixel-counter">
    <span class="pixel-bar">■■■</span>
    <span id="keywords-count" class="favorites-count">0</span>
    <span class="pixel-bar">■■■</span>
  </div>
</div>

<div class="keywords-add-card">
  <h3 class="keywords-section-subtitle">ДОБАВИТЬ</h3>
  <div class="keywords-input-container">
    <div class="keywords-input-wrapper">
      <input type="text" id="new-keyword-input" placeholder="Введите ключевое слово...">
      <button id="keywords-clear-button" class="keywords-clear-button" aria-label="Очистить"></button>
    </div>
    <button id="add-keyword-btn" class="keywords-add-button">+</button>
  </div>
</div>
```

## CSS классы
- `.keywords-window-header` - заголовок окна с синим фоном
- `.keywords-window-title` - заголовок "КЛЮЧЕВЫЕ СЛОВА"
- `.pixel-counter` - контейнер счетчика в пиксель-стиле
- `.pixel-bar` - декоративные полоски по бокам счетчика (с анимацией)
- `.favorites-count` - сам счетчик с красным фоном
- `.keywords-input-wrapper` - обёртка поля ввода для позиционирования крестика
- `.keywords-input-wrapper.has-text` - состояние при наличии текста (показывает крестик)
- `.keywords-clear-button` - кнопка очистки с псевдоэлементами (скрыта по умолчанию)

## JavaScript функции
- `updateKeywordsInputState()` - обновляет состояние крестика
- `addKeyword(keyword)` - добавляет ключевое слово
- `removeKeyword(keyword)` - удаляет ключевое слово
- `updateKeywordsInDatabase()` - сохраняет в Supabase
- `loadKeywords()` - загружает из Supabase
