### Критичная проблема
- **Maximum call stack size exceeded** - бесконечная рекурсия в JavaScript
- Ломалась структура приложения при навигации между страницами
- Слетал дизайн после выхода из настроек/избранного

### Причина
Дублирование инициализации в JavaScript модулях:
1. `theme-manager.js` - двойная инициализация `initTheme()`
2. `template-loader.js` - дублирование загрузки шаблонов
3. HTML файлы - дублирование CSS и скриптов
